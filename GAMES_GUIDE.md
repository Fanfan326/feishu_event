## 💣 工具2：单词炸弹（挑战模式）

**特点**：
- ⏰ 限时答题（超时炸弹爆炸！）
- 🔥 连击系统（连续答对加分）
- ❤️ 生命值系统（3条命）
- 👾 Boss关卡（超级单词挑战）
- 🏆 成就系统（解锁成就）

**启动方式**：
```bash
# 方式1：快捷命令
~/wordbomb

# 方式2：完整路径
bash /Users/francinapeng/Public/start_word_bomb.sh

# 方式3：直接运行
python3 /Users/francinapeng/Public/word_bomb.py
```

**游戏规则**：
1. 每个单词有倒计时（简单10秒、中等8秒、困难6秒）
2. 时间内回答正确得分，超时炸弹爆炸扣生命
3. 连续答对有连击加分
4. 每3关有Boss挑战（超长单词）
5. 3条生命用完游戏结束

**评分系统**：
- 基础分：10分
- 时间奖励：剩余时间 × 10
- 连击加成：连击数 × 5
- Boss战：拼写正确+100分，中文意思+50分

**成就系统**：
- 🏆 5连击
- 🏆 10连击 - 连击高手
- 🏆 100分 - 百分小子
- 🏆 500分 - 单词大师

**评级标准**：
- S级：500分+ 单词大师
- A级：300-499分 词汇高手
- B级：150-299分 不错哦
- C级：0-149分 继续努力

## 🎯 使用建议

### 学习路径1：从零开始
1. 先用 **词汇训练工具** 学习单词（快速学习模式）
2. 熟悉后玩 **单词炸弹** 挑战自己
3. 每天复习（词汇训练工具 → 今日复习）

### 学习路径2：刺激挑战
1. 直接玩 **单词炸弹**
2. 遇到不会的词，用 **词汇训练工具** 查询
3. 学完后再挑战

### 每日建议时间
- 早上：词汇训练工具（10分钟）
- 中午：单词炸弹快速一局（5分钟）
- 晚上：复习（5分钟）

**总计每天20分钟，一个月后词汇量爆炸式增长！** 🚀

## 💡 小技巧

### 单词炸弹技巧
1. **时间管理**：简单的词快速作答，省时间冲分
2. **连击至上**：保持连击比单题高分更重要
3. **Boss战略**：
   - 自信就选拼写（+100分）
   - 不确定选中文（+50分保底）
4. **生命管理**：3条命要合理分配，不要前期浪费

### 学习技巧
1. 先用词汇训练工具看单词拆解和记忆方法
2. 立刻用单词炸弹实战测试
3. 错的单词回到训练工具复习
4. 形成学习闭环
